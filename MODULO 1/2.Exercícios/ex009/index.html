Trabalhando com Listas

Listas Ordenadas

São listas criadas com a TAG <ol> que tem o nome de ordered lists (Listas Ordenadas)que tem um padrão de organização.
Dentro dela tambem temos a TAG <li> que tem o nome de list item (Listar Item) para indentificar cada item da lista.

Usando o parâmetro type, configuramos o tipo de marcador da lista: 1, A, a, I, i.

Usando o parâmetro start, pode in dicar o início da contagem
Por exemplo, a tag <ol type=“1” start = “5”> vai gerar itens numerados como 5.
5, 6, 7, 8, …

  1. Acordar
  2. Tomar Café
  3. Escovar os Dentes
  4. Ir para a faculdade
  5. Almoçãr

Listas Não Ordenadas

São listas criadas com a TAG <ul> que tem o nome de unordered lists (Listas Não Ordenadas), ela não possui uma ordem, pode ser criada da maneira que quiser.

Dentro dela tamvém possui a TAG <li>./p>

Usando o parâmetro type podemos escolher um marcador, com os seguintes valores.

disc - padrão. Uma bola preta totalmente pintada
circle - Uma bola com uma borda preta e sem preenchimento
square - Um pequeno quadrado preto totalmente pintado

  • Pão
  • Leite
  • Tomate
  • Arroz
